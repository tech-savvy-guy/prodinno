# ProdInno Website Template

A modern website template built with Next.js 14, TypeScript, and Tailwind CSS. Features dark theme, 3D graphics, animations, and responsive design.

## Getting Started

1. Clone the repository
   ```bash
   git clone https://github.com/tech-savvy-guy/prodinno.git
   cd prodinno
   ```

2. Install dependencies
   ```bash
   pnpm install
   ```

3. Run development server
   ```bash
   pnpm dev
   ```

4. Open [http://localhost:3000](http://localhost:3000)

## Scripts

- `pnpm dev` - Development server
- `pnpm build` - Production build
- `pnpm start` - Production server

## Pages

- Home - `/`
